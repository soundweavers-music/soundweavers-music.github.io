## 介紹

Array mbira是一種手工製作的現代樂器，具有獨特的豎琴或鐘聲般的音質。它由其發明者比爾·韋斯利在美國製造，並由韋斯利與帕特里克·哈德利在美國加州聖地亞哥合作生產。它的開發始於1960年代。它是對來自津巴布韋肖納人非洲mbira樂器的徹底重新設計，是金屬舌簧樂器家族的一部分。金屬舌片被分組排列成多個八度音階。

### 代表性作品

由於Array mbira是1960年代才發明的現代樂器，專為其創作的代表作品相對較少。該樂器主要應用於當代實驗音樂、環境音樂創作，以及融合傳統非洲音樂元素的現代作品中。許多作曲家受到其獨特音色啟發，將其融入室內樂、電子音樂混音作品。韋斯利本人與哈德利在聖地亞哥進行的多次現場演奏與錄音，成為該樂器的重要文獻記錄。

## 歷史背景

Array mbira是一種現代手工製作樂器，源於1960年代美國發明家比爾·韋斯利的創意構想。韋斯利受到津巴布韋肖納人傳統mbira樂器的啟發，對其進行徹底重新設計，創造出具有豎琴般音質的全新樂器。Array mbira屬於金屬舌簧樂器家族，其獨特之處在於將金屬舌片分組排列成多個八度音階，產生獨特的音響效果。該樂器由韋斯利與帕特里克·哈德利在美國加州聖地亞哥合作生產。從津巴布韋傳統樂器的東非根源出發，經由美國音樂創新者的改良設計，Array mbira逐漸在西方現代音樂領域傳播，成為一種融合東西方音樂文化特色的當代樂器，展現了全球化時代傳統樂器的創新發展脈絡。

## 音色描述


Array mbira的音色質感明亮而透澈，帶有獨特的金屬質感與溫潤的諧音層次。其音域跨越多個八度，能夠表現從低沉的基音到清晰的高音，具有良好的音域寬度。樂器採用金屬舌片設計，每次撥彈產生的振動在共鳴體內產生豐富的泛音，形成類似豎琴的柔和光澤感與鐘樂般的清越音質。其共鳴方式融合了傳統非洲mbira的溫暖深度與現代製造的精準音準，兼具打擊樂器的穿透力與弦樂器的持續感。相較於傳統鋼琴的厚重感，Array mbira音色更為輕盈靈動；相比吉他的溫暖，其音質更具金屬光澤與空靈特質。
## 樂器材質

Array mbira採用多層結構設計，共鳴體主要由優質木材製成，提供溫暖的音響特性。金屬舌片採用經過精密加工的鋼或磷青銅製造，確保音準與耐用性。舌片分組排列以實現多個八度音階的設計。框架與支撐結構採用金屬合金材料，增強樂器的結構穩定性與音響投射能力。部分版本可能使用皮革或布料作為減振材料。整體工藝融合傳統手工製作與現代精密製造技術。

## 教學

手持樂器，以敲擊方式發聲。手持樂器本體，透過敲擊、搖晃、刮奏或撥彈等方式使樂器本身震動發聲。不同部位和不同力道會產生不同音色。練習時先確認正確持琴姿勢，掌握基本發聲方式後再練習節奏型和旋律。用指尖或撥子撥奏弦線，注意觸弦角度和力度控制。
